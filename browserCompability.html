Browser Compability

What is browser compability?

There are different web browsers in use and all of them read the code a litte bit differently. When you make a website it might not look all the same in every browser. You want your website to be flexible and to funtion for any user.

Why is browser compability important?

Browser compability is important because you want anyone to be able to access your website. It needs to function properly and look the best in any browser and on any device. Users will access websites from mobiles, tablets and computers and their web browser is not always updated to the latest version.
